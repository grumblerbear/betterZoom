#betterZoom

##Options: 

###width
Default: **200**  
Possible values: **integer**  
Ширина лупы

###height
Default: **200**  
Possible values: **integer**  
Высота лупы

###limits
Default: **true**  
Possible values: **true / false**  
Выход лупы за пределы картинки

###overlay
Default: **false**  
Possible values: **true / false**  
Наложение оверлея на картинку
